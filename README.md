# 🏗️ BEAM SUPPORT

Aplicación web desarrollada en **React** que permite calcular la resistencia de una viga sometida a carga puntual, facilitando el análisis estructural de forma rápida e intuitiva.

---

## Funcionalidades

- Cálculo de resistencia de vigas
- Ingreso de datos:
  - Longitud de la viga
  - Carga puntual
  - Módulo de resistencia
  - Esfuerzo admisible
- Resultados inmediatos
- Interfaz intuitiva
